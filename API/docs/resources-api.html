Basic Resource Operations

CRUD operations for managing resources in the PMO system

GET /resources
Retrieve all resources with their details

Fetches all resources from the PMO system including personal details, roles, capacity, and organizational information.

Response Fields

Field Type Description
resource_id integer Unique resource identifier
resource_name string Full name of the resource
resource_email string Email address
resource_type string Employee type (Full-time, Contractor, etc.)
strategic_portfolio string Business portfolio assignment
product_line string Product line assignment
manager_name string Direct manager's name
resource_role string Job role/title
yearly_capacity number Annual working hours capacity
skillset string Technical and professional skills

PowerShell Example

# Get all resources
$response = Invoke-WebRequest -Uri "http://localhost:5000/resources" -Method GET
$resources = $response.Content | ConvertFrom-Json

# Display results
$resources | Format-Table resource_id, resource_name, resource_role, strategic_portfolio

# Count resources
Write-Host "Total resources: $($resources.Count)"

Response Example

  }
]
GET /resources/{resource_id}
Retrieve a specific resource by ID

Fetches detailed information for a single resource using their unique ID.

Path Parameters

Parameter Type Required Description
resource_id integer Yes Unique identifier for the resource

PowerShell Example

# Get specific resource
$resourceId = 101
$response = Invoke-WebRequest -Uri "http://localhost:5000/resources/$resourceId" -Method GET
$resource = $response.Content | ConvertFrom-Json

# Display resource details
$resource | Format-List
POST /resources
Add a new resource to the system

Creates a new resource record with all necessary details for project allocation and capacity planning.

Request Body Schema

Field Type Required Description
resource_name string Yes Full name of the resource
resource_email string Yes Email address
resource_role string Yes Job role/title
yearly_capacity number Yes Annual working hours
strategic_portfolio string No Business portfolio
skillset string No Skills and competencies

PowerShell Example

# Create new resource
$body = @{
    resource_name = "Alice Developer"
    resource_email = "alice.dev@company.com"
    resource_type = "Full-time"
    strategic_portfolio = "Technology"
    product_line = "Software Development"
    manager_name = "Bob Manager"
    manager_email = "bob.manager@company.com"
    resource_role = "Frontend Developer"
    responsibility = "UI/UX Development"
    skillset = "React, TypeScript, CSS"
    yearly_capacity = 2088
    timesheet_resource_name = "A.Developer"
    comments = "Specializes in modern web frameworks"
} | ConvertTo-Json

$response = Invoke-WebRequest -Uri "http://localhost:5000/resources" -Method POST -Body $body -ContentType "application/json"
$result = $response.Content | ConvertFrom-Json

Write-Host "Resource added: $($result.message)"

Capacity Management

Advanced endpoints for tracking resource capacity, availability, and workload planning

GET /resource_capacity
Get resource capacity over time periods

Retrieves resource capacity data for planning and analysis. Supports weekly, monthly, or block-based intervals with time-off adjustments.

Query Parameters

Parameter Type Required Description
resource_id integer Yes Resource identifier
start_date date No Start date (YYYY-MM-DD), defaults to current year start
end_date date No End date (YYYY-MM-DD), defaults to current year end
interval string No Weekly, Monthly, or empty for blocks (default: Monthly)

PowerShell Example

# Get monthly capacity for resource
$params = @{
    resource_id = 101
    start_date = "2024-01-01"
    end_date = "2024-12-31"
    interval = "Monthly"
}

$queryString = ($params.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join "&"
$uri = "http://localhost:5000/resource_capacity?$queryString"

$response = Invoke-WebRequest -Uri $uri -Method GET
$capacity = $response.Content | ConvertFrom-Json

# Display capacity summary
$capacity.data | Format-Table start_date, end_date, total_capacity, available_capacity

Response Example

GET /resource_capacity_allocation
Get resource capacity with project allocation details

Comprehensive endpoint that combines capacity data with detailed project allocation information, including planned vs actual hours and project breakdown.

Query Parameters

Parameter Type Required Description
resource_id string Yes Resource identifier
start_date string No Start date (YYYY-MM-DD)
end_date string No End date (YYYY-MM-DD)
interval string No Weekly, Monthly, or empty for blocks
project_id integer No Filter by specific project

PowerShell Example

# Get resource allocation with project details
$params = @{
    resource_id = "101"
    start_date = "2024-01-01"
    end_date = "2024-06-30"
    interval = "Monthly"
}

$queryString = ($params.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join "&"
$uri = "http://localhost:5000/resource_capacity_allocation?$queryString"

$response = Invoke-WebRequest -Uri $uri -Method GET
$allocation = $response.Content | ConvertFrom-Json

# Display allocation summary
foreach ($period in $allocation.data) {
    Write-Host "Period: $($period.start_date) to $($period.end_date)"
    Write-Host "  Total Capacity: $($period.total_capacity) hours"
    Write-Host "  Projects:"
    
    foreach ($project in $period.project_allocation_details) {
        Write-Host "    - $($project.project_name): $($project.planned_hours)h planned, $($project.actual_hours)h actual"
    }
    Write-Host ""
}
GET /resource_capacity_allocation_per_portfolio
Get aggregated capacity data by portfolio

Aggregates resource capacity and allocation data across multiple resources filtered by strategic portfolio and/or product line.

Query Parameters

Parameter Type Required Description
strategic_portfolio string No Filter by strategic portfolio
product_line string No Filter by product line
start_date string No Start date (YYYY-MM-DD)
end_date string No End date (YYYY-MM-DD)
interval string No Weekly, Monthly, or empty for blocks

PowerShell Example

# Get portfolio capacity overview
$params = @{
    strategic_portfolio = "Technology"
    interval = "Monthly"
    start_date = "2024-01-01"
    end_date = "2024-12-31"
}

$queryString = ($params.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join "&"
$uri = "http://localhost:5000/resource_capacity_allocation_per_portfolio?$queryString"

$response = Invoke-WebRequest -Uri $uri -Method GET
$portfolioData = $response.Content | ConvertFrom-Json

# Display portfolio summary
Write-Host "Portfolio: $($portfolioData.strategic_portfolio)"
Write-Host "Grand Total Capacity: $($portfolioData.grand_total_capacity) hours"
Write-Host "Grand Total Planned: $($portfolioData.grand_total_allocation_hours_planned) hours"
Write-Host "Grand Total Available: $($portfolioData.grand_total_available_capacity) hours"

# Show resource breakdown
Write-Host "`nResource Summary:"
$portfolioData.resource_summary | Format-Table resource_name, resource_role, total_capacity, allocation_hours_planned, available_capacity

Allocation Tracking

Project-specific resource allocation and capacity tracking endpoints

GET /resource_capacity_allocation_by_project
Get resource allocation broken down by projects

Provides detailed breakdown of how a resource's capacity is allocated across different projects over time periods.

Query Parameters

Parameter Type Required Description
resource_id string Yes Resource identifier
start_date string No Start date (YYYY-MM-DD)
end_date string No End date (YYYY-MM-DD)
interval string No Weekly, Monthly, or empty for blocks

PowerShell Example

# Get project breakdown for resource
$params = @{
    resource_id = "101"
    start_date = "2024-01-01"
    end_date = "2024-06-30"
    interval = "Monthly"
}

$queryString = ($params.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join "&"
$uri = "http://localhost:5000/resource_capacity_allocation_by_project?$queryString"

$response = Invoke-WebRequest -Uri $uri -Method GET
$projectData = $response.Content | ConvertFrom-Json

# Display project allocation breakdown
Write-Host "Resource: $($projectData.resource_details.resource_name)"
Write-Host ""

foreach ($interval in $projectData.intervals) {
    Write-Host "Period: $($interval.start_date) to $($interval.end_date)"
    
    foreach ($project in $interval.projects) {
        $utilization = [math]::Round(($project.allocation_hours_planned / $project.total_capacity) * 100, 1)
        Write-Host "  $($project.project_name):"
        Write-Host "    Planned: $($project.allocation_hours_planned)h ($utilization%)"
        Write-Host "    Actual: $($project.allocation_hours_actual)h"
        Write-Host "    Cost: `$$($project.allocation_cost_planned)"
    }
    Write-Host ""
}
GET /project_capacity_allocation/{project_id}
Get capacity allocation for a specific project

Aggregates capacity and allocation data for all resources assigned to a specific project, providing project-centric view of resource utilization.

Path Parameters

Parameter Type Required Description
project_id integer Yes Project identifier

Query Parameters

Parameter Type Required Description
interval string No Weekly, Monthly, or empty for blocks
start_date string No Override project start date
end_date string No Override project end date

PowerShell Example

# Get project capacity allocation
$projectId = 202
$params = @{
    interval = "Monthly"
}

$queryString = ($params.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join "&"
$uri = "http://localhost:5000/project_capacity_allocation/$projectId"
if ($queryString) { $uri += "?$queryString" }

$response = Invoke-WebRequest -Uri $uri -Method GET
$projectCapacity = $response.Content | ConvertFrom-Json

# Display project summary
Write-Host "Project: $($projectCapacity.project_name)"
Write-Host "Portfolio: $($projectCapacity.project_strategic_portfolio)"
Write-Host "Duration: $($projectCapacity.start_date_est) to $($projectCapacity.end_date_est)"
Write-Host ""
Write-Host "Grand Totals:"
Write-Host "  Total Capacity: $($projectCapacity.grand_total_capacity) hours"
Write-Host "  Planned Hours: $($projectCapacity.grand_total_allocation_hours_planned) hours"
Write-Host "  Available Hours: $($projectCapacity.grand_total_available_capacity) hours"

# Show resource details by period
foreach ($interval in $projectCapacity.intervals) {
    Write-Host "`nPeriod: $($interval.start_date) to $($interval.end_date)"
    Write-Host "Resources:"
    
    $interval.resource_details | Format-Table resource_name, resource_role, allocation_hours_planned, allocation_hours_actual, allocation_cost_planned
}
