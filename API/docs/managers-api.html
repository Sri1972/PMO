Managers API

Simple and efficient manager directory management for organizational hierarchy and contact information.

1 Endpoint
3 Data Fields
Simple Design

Manager Endpoints

Access manager directory and contact information

GET /managers Get all managers with contact information and portfolio assignments

Description

Retrieves a complete list of all managers including their contact information and strategic portfolio assignments. Results are ordered alphabetically by manager name.

Response Fields

Field Type Description
manager_name string Full name of the manager
manager_email string Manager's email address
strategic_portfolio string Strategic portfolio managed by this manager

Response Example

[
  {
    "manager_name": "Alice Johnson",
    "manager_email": "alice.johnson@company.com",
    "strategic_portfolio": "Market & Sell"
  },
  {
    "manager_name": "Bob Smith",
    "manager_email": "bob.smith@company.com",
    "strategic_portfolio": "Auto Insights"
  },
  {
    "manager_name": "Carol Davis",
    "manager_email": "carol.davis@company.com",
    "strategic_portfolio": "Commercial Operations"
  }

PowerShell Example

Invoke-WebRequest -Uri "http://localhost:5000/managers" `
  -Method GET `
  -Headers @{"Content-Type"="application/json"}

cURL Example

curl -X GET "http://localhost:5000/managers" \
  -H "Content-Type: application/json"

Common Use Cases

Practical applications for the Managers API

Manager Directory

Build a manager contact directory for organizational reference

# Get all managers for directory
$managers = (Invoke-WebRequest -Uri "http://localhost:5000/managers" `
  -Method GET `
  -Headers @{"Content-Type"="application/json"}).Content | ConvertFrom-Json

# Display formatted directory
$managers | Format-Table manager_name, manager_email, strategic_portfolio

Portfolio Management

Find managers responsible for specific portfolios

# Get managers and filter by portfolio
$managers = (Invoke-WebRequest -Uri "http://localhost:5000/managers" `
  -Method GET `
  -Headers @{"Content-Type"="application/json"}).Content | ConvertFrom-Json

# Find Market & Sell managers
$marketSellManagers = $managers | Where-Object { $_.strategic_portfolio -eq "Market & Sell" }

Contact Integration

Extract email addresses for communication workflows

# Get all manager emails
$managers = (Invoke-WebRequest -Uri "http://localhost:5000/managers" `
  -Method GET `
  -Headers @{"Content-Type"="application/json"}).Content | ConvertFrom-Json

# Extract email list
$managerEmails = $managers | Select-Object -ExpandProperty manager_email
$emailList = $managerEmails -join ";"

Integration with Other APIs

How the Managers API works with other PMO endpoints

Resources API

Resources include manager information - use this API to get the complete manager directory.

# Cross-reference resource managers
$resources = (Invoke-WebRequest -Uri "http://localhost:5000/resources").Content | ConvertFrom-Json
$managers = (Invoke-WebRequest -Uri "http://localhost:5000/managers").Content | ConvertFrom-Json

# Find manager details for a resource
$resourceManager = $managers | Where-Object { $_.manager_email -eq $resources[0].manager_email }

Business Lines API

Combine manager data with business line structure for organizational mapping.

# Map managers to business lines
$managers = (Invoke-WebRequest -Uri "http://localhost:5000/managers").Content | ConvertFrom-Json
$businessLines = (Invoke-WebRequest -Uri "http://localhost:5000/business_lines").Content | ConvertFrom-Json

# Group by strategic portfolio
$organizationMap = $managers | Group-Object strategic_portfolio
